### Installation

1. Clone the repository
```bash
git clone https://github.com/adrii-x/Video-flow.git
```

2. Install dependencies
```bash
cd Video-flow
npm install
```

3. Start the development server
```bash
npm run dev
```

4. Open your browser and navigate to `http://localhost:8080`
